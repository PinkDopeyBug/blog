**使用数据库或创建数据库**
如果要使用的数据库不存在则创建
```
use 数据库名字
```
创建数据库操作先是创建在内存中,并没有创建到硬盘中,此时使用查询将查询不到新建的数据库,如果对新建的数据库进行插入数据的操作时数据库会自动保存到硬盘中了
> [!warning] 注意
> 在MongoDB中，集合只有在内容插入后才会创建！就是说。创建集合(数据表)后要再插入一个文档（记录），集合才会真正创建。

**查看有权限查看的所有数据库命令**
```
show dbs
或
show databases
```

**查看当前正在使用的数据库命令**
```
db
```

**删除数据库**
删除正在使用的数据库
```
db.dropDatebase()
```
